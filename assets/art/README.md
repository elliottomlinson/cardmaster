art

│   └── <location-name>..........folders containing art for each location

├── item.....................item card artwork
│   ├── back.....................backs for each item deck called <location>
│   └── <location-name>..........folders with item art for each location's deck
└── misc.....................other artwork and art resources
    ├── template.................templates for all art types
    └── <tool-name>..............folders for art made for tts tools
